Installation

From Packages

For RPM-based distributions (Fedora, RHEL):

sudo rpm -i tuxcut-qt-*.rpm

For DEB-based distributions (Ubuntu, Debian):

sudo dpkg -i tuxcut-qt-*.deb

From Source

git clone https://github.com/se7uh/tuxcut-qt cd tuxcut-qt python -m venv .venv source .venv/bin/activate pip install -r requirements.txt sudo .venv/bin/python client/tuxcut_qt.py

About

TuxCut Qt is a modern network management tool for Linux systems. It provides an intuitive interface for controlling network access, managing devices, and protecting your computer.
